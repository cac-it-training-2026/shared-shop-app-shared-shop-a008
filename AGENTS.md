# ECサイト（Shared Shop）

## システム概要

Spring Bootを用いて開発したECサイトです。

一般会員は商品閲覧・カート利用・注文・会員情報管理を行うことができ、管理者は商品・カテゴリ・会員・注文情報の管理を行うことができます。

### 主な特徴

- 会員認証機能
- 商品管理機能
- ショッピングカート機能
- 注文管理機能
- 管理者機能
- 商品画像アップロード機能
- 入力バリデーション機能

## データベース構成

### users（会員情報）

| カラム | 内容 |
|---------|---------|
| id | 会員ID |
| name | 氏名 |
| email | メールアドレス |
| password | パスワード |
| address | 住所 |

### categories（カテゴリ情報）

| カラム | 内容 |
|---------|---------|
| id | カテゴリID |
| name | カテゴリ名 |

### items（商品情報）

| カラム | 内容 |
|---------|---------|

# 開発時の禁止事項

* pom.xmlの変更を生じる新たなライブラリの導入は禁止（フロントエンドのCDNはOK）
* サービスレイヤは含まず、基礎的なＭＶＣで構成する（Controllerから直接JPAリポジトリのメソッドを呼んでいる）
* コーディング規約はクラスはUpperキャメル、メソッドはlowerキャメルなど原則、Google Java Styleに準拠する
---
# Julesのルール

* セッション内のやり取り、プルリクエストの内容は日本語で記述して下さい
* 画面に変更がある場合はマルチモーダルで視覚的に示してください
